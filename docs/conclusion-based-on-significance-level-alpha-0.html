# **How to Perform Two-Way ANOVA in Python and R using the iris dataset?**

## **Explanation**  
**Two-Way ANOVA** is used to examine the effect of two independent variables on a dependent variable. In this case, we will explore how both the **species** and **sepal_width** of the iris flowers affect the **sepal_length**.

### **Hypotheses:**  
- **H0 (Null Hypothesis):** There is no significant effect of species, sepal width, or their interaction on the sepal length.  
- **H1 (Alternative Hypothesis):** There is a significant effect of species, sepal width, or their interaction on the sepal length.

### **Dataset:**  
We will use the `iris` dataset with columns: `sepal_length`, `sepal_width`, `petal_length`, `petal_width`, and `species`, stored in `data/iris.csv`.

---

## **Python Code**


```python
import pandas as pd
import statsmodels.api as sm
from statsmodels.formula.api import ols
from statsmodels.stats.anova import anova_lm

# Load the iris dataset
df = pd.read_csv("data/iris.csv")

# Perform Two-Way ANOVA (considering species and sepal_width as independent variables)
model = ols('sepal_length ~ species + sepal_width + species:sepal_width', data=df).fit()

# Get the ANOVA table
anova_result = anova_lm(model)

# Display results
print(anova_result)

# Interpretation
if anova_result["PR(>F)"][0] < 0.05:
    print("Reject the null hypothesis: There is a significant effect of species on sepal length.")
else:
    print("Fail to reject the null hypothesis: No significant effect of species on sepal length.")

if anova_result["PR(>F)"][1] < 0.05:
    print("Reject the null hypothesis: There is a significant effect of sepal width on sepal length.")
else:
    print("Fail to reject the null hypothesis: No significant effect of sepal width on sepal length.")

if anova_result["PR(>F)"][2] < 0.05:
    print("Reject the null hypothesis: There is a significant interaction between species and sepal width on sepal length.")
else:
    print("Fail to reject the null hypothesis: No significant interaction between species and sepal width on sepal length.")
